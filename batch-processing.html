Batch chromatogram processing

Peaksel allows you to apply changes to many chromatograms at a time: change integration parameters, manually select or update peaks, align detectors in all injections, etc. The most suspicious samples are then highlighted so that you don't have to go through all of them for manual curation.

Wondering if you missed 1 sample out of hundred? QC them quickly and find the outliers

Still want to go through all the chromatograms? Use W, A, S, D keys to quickly navigate between them
