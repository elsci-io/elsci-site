Manual upload of raw data

Most scientists prefer to set up an automatic import of raw data from the instruments, but if you want to play with some injections and just quickly get the results, Peaksel allows uploading data manually as well. For this you simply open Analyze new injections page and drag'n'drop your zip with raw data.

Depending on a vendor, zip archive should contain:

  • Waters: a directory or multiple directories with extension RAW, like this: 001.RAW
  • Agilent: a directory with the sequence, inside of it there should be multiple .D folders.
  • MzML: a directory or multiple directories with mzml extension, like this: 001.mzml. Inside the directory there should be one or more files for each detector with *.MzML extension.

By default ZIP can't be greater than 1GB in size. The default can be changed in the Global Properties. Alternatively, you can set up automatic import.
