Isotopic patterns, qualifier and quantifier ions

It's possible to fine-tune which of the ions that Peaksel calculated should be used. For instance, you may want to use one set of ions as quantifiers (they'll be part of the peak area), while others are used as qualifiers to determine if the peak is actually our compound and not some isobar with very similar mass.

By default, Peaksel will use only the most abundant isotopes (of the same adduct!) as qualifiers, but only if their predicted intensities are higher than the tolerance level.
