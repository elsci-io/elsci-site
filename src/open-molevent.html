Open Molecule Event Database

You probably have to frequently deal with a lot of open databases. Imagine a single website where the information from all those databases is aggregated. We have a good understanding of how to make it possible.

We're looking for companies who're willing to sponsor development of such database. We already have some capabilities implemented as part of our Molevent, but a lot of work is still needed. There are 2 parts to it:

  1. It's going to be a huge database, so we need to implement a technical possibility to handle that many records.
  2. And we need to write ETLs that would gather information from other sources.

If you're interested in sponsoring or want to be notified about the progress, let us know at support@elsci.io.
